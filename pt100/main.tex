\begin{abstract}

\section{Model}

For the PT-100, we assume a linear relation between celsius temperature $T$ and 
output voltage $V$ given by
\begin{align}
    T &= mV + k \\
    \implies \vec{\theta}^\top\vec{x} &= y \label{eq:model}
\end{align}
where
\begin{align}
    y = T,\ \vec{\theta} = \myvec{m\\k},\ \vec{x} = \myvec{V\\1}
    \label{eq:x-y-theta-def}
\end{align}

\section{Optimization}
To find the optimal parameters of the model \eqref{eq:model} can now be modeled 
as an unconstrained optimization problem \eqref{eq:opt-model}
\begin{align}
    \min_{\vec{\theta}}\sum_{i}\abs{\vec{\theta}^\top x_i - y_i}^2
    \label{eq:opt-model}
\end{align}
which is solved by the Python code \texttt{codes/lsq.py}.

The optimized parameters are
\begin{align}
    \vec{\theta} = \myvec{333.05\\-551.34}
    \label{eq:opt-theta}
\end{align}
